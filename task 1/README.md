# Personal Portfolio Project

### [  💻 Project Overview  ]
This is a single-page personal portfolio application developed to showcase technical skills and project work, featuring smooth navigation and interactive elements.

---

### [ ✨ Key Features ]

1.  **Direct Section Navigation:** Responsive navigation bar with anchor links for smooth scrolling to specific page sections.
2.  **Interactive Interface:** Modern UI elements with hover effects on buttons and table rows for better user engagement.
3.  **Validate & Capture Form:** Client-side JavaScript validation ensures correct user input before submission.
