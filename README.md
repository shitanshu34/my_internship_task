# Personal Portfolio Project

This is my first internship task involving the creation of a professional portfolio website.

## 🚀 Technologies Used
* **HTML5**: For semantic structure.
* **CSS3**: For professional styling, Box Model, and interactive hover effects.
* **JavaScript**: For DOM manipulation, event handling, and form validation.

## ✨ Key Features
* **Navigation**: Clickable anchor links to scroll between sections.
* **Skills Table**: Technical skills organized with icons and hover highlighting.
* **Contact Form**: Functional form with validation for Name, Email, and Subject.
* **Responsive Design**: Basic layout structured for clarity.

## 📂 Project Files
* `portfolio.html` - Main structure
* `style.css` - Custom professional styling
* `script.js` - Form logic and interactivity
